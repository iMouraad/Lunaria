# Lunaria — Perfumería de Lujo Ecuador

Sitio web de LUNARIA, perfumería 100% original en Ecuador. Fragancias de Diseñador, Árabes, Nicho y Frescos con envíos a todo el país.

## Tecnologías
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Páginas
- `index.html` — Página principal
- `designer.html` — Colección Diseñador
- `arabes.html` — Colección Árabes
- `nicho.html` — Colección Nicho (próximamente)
- `producto.html` — Detalle de producto

## Características
- Buscador en tiempo real de fragancias
- Carrito de compras con redirección a WhatsApp
- Selector de tamaños (Muestra / Tamaño completo)
- Envío gratuito desde $100
